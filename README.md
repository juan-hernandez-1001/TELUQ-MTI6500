# TELUQ-MTI6500
This repository contains the files related to the MTI6500 course.
It contains the 528 datasets needed to perform the 528 multiple linear regressions performed for this course, along with the corresponding regression results.
File names indicate the Canadian stock maket index whose data they contain.
Files whose name includes "pre struc break" contain data related to the period preceding the structural break.
Files whose name includes "post struc break" contain data related to the period following the structural break.
Sheets whose name includes "Eq4" show the contemporaneous relationship between the sentiment variable in column E of the sheet and the return of the corresponding index.
Sheets whose name includes "Eq7" show the lead-lag relationship between the sentiment variable in column E of the sheet and the return of the corresponding index.
Sheet "Sommaire" contains a summary of the coefficients of the contemporaneous relationships (column EV) and the lead-lag relationships (column EVl1) between the sentiment variables and the return of the corresponding index.
